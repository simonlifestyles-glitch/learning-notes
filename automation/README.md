## 一、什麼是自動化腳本？

**腳本（Script）** 是一份「指令清單」，告訴電腦按順序執行一連串動作。

想像你每天上班都要做同樣的事：整理檔案、發報告、抓資料⋯⋯這些重複的工作，可以用**程式代替你做**，這就是自動化腳本的核心概念。

> 💡 **一句話總結：你做過一次的重複工作，都可以讓程式幫你做。**

---

## 二、Python vs Shell — 有什麼不同？

| | Python | Shell（Bash） |
|---|---|---|
| **語法風格** | 接近英文句子，易讀 | 短指令組合，簡潔 |
| **擅長領域** | 資料處理、邏輯運算、爬蟲 | 操作檔案、系統管理 |
| **適合任務** | 複雜任務 | 簡單快速的系統操作 |
| **平台支援** | Windows / Mac / Linux | 主要 Mac / Linux |

**簡單區分：**
- Shell → 跟**作業系統**說話
- Python → 處理**更複雜的邏輯**

## 三、生活化的例子

### 🐚 Shell 腳本範例 — 自動備份資料夾

```bash
# 每天自動備份資料夾，並加上當天日期
cp -r ~/Documents/工作 ~/備份/工作_$(date +%Y%m%d)
echo "備份完成！"
```

**說明：** 這段腳本會把「工作」資料夾複製一份，並在資料夾名稱後加上今天的日期（例如 `工作_20240101`）。

---

### 🐍 Python 腳本範例 — 自動批次改名

```python
import os

# 自動把所有 .txt 檔改名，加上「已處理_」前綴
for 檔案 in os.listdir("./資料夾"):
    if 檔案.endswith(".txt"):
        os.rename(檔案, "已處理_" + 檔案)

print("全部改名完成！")
```

**說明：** 這段腳本會掃描指定資料夾，把所有 `.txt` 文字檔自動加上「已處理_」的前綴。

## 六、實際操作步驟

### 🐚 Shell — 用 Bash 執行腳本

**步驟一：開啟終端機**
- Mac：按 `Command + 空白鍵`，搜尋 `終端機`
- Linux：搜尋 `Terminal`
- Windows：安裝 `Git Bash` 後開啟

**步驟二：建立腳本檔案**
```bash
# 建立一個新的 .sh 檔
touch hello.sh
```

**步驟三：用文字編輯器寫入內容**
```bash
# 用內建編輯器開啟（按 i 進入編輯，按 Esc 後輸入 :wq 儲存離開）
vi hello.sh
```

腳本內容範例：
```bash
#!/bin/bash
echo "哈囉，這是我的第一個 Shell 腳本！"
echo "今天是：$(date)"
```

> 📌 第一行 `#!/bin/bash` 稱為 **Shebang**，告訴系統用 Bash 來執行這個檔案，**每個 .sh 檔都要加**。

**步驟四：給予執行權限並執行**
```bash
# 賦予執行權限（只需做一次）
chmod +x hello.sh

# 執行腳本
./hello.sh

# 或是直接用 bash 執行（不需要權限）
bash hello.sh
```

**預期輸出：**
```
哈囉，這是我的第一個 Shell 腳本！
今天是：Sun Mar 22 10:00:00 CST 2026
```

### 🐍 Python — 用 VS Code 執行腳本

**步驟一：安裝必要工具**

| 工具 | 下載網址 | 說明 |
|---|---|---|
| Python | `python.org` | 安裝時記得勾選 **Add to PATH** |
| VS Code | `code.visualstudio.com` | 程式碼編輯器 |
| Python 擴充套件 | VS Code 內搜尋 `Python` | Microsoft 官方出品，必裝 |

**步驟二：建立 .py 檔案**

在 VS Code 中按 `Ctrl + N`（Mac 用 `Cmd + N`）新增檔案，另存為 `hello.py`。

**步驟三：寫入程式碼**
```python
# 我的第一個 Python 腳本
print("哈囉，這是我的第一個 Python 腳本！")

name = input("請輸入你的名字：")
print(f"嗨，{name}！歡迎來到 Python 的世界 🎉")
```

**步驟四：執行**

有兩種方式：

```
方式 A：點右上角 ▶ 播放鍵  ← 最簡單
方式 B：在 VS Code 下方 Terminal 輸入：
        python3 hello.py
```

**預期輸出：**
```
哈囉，這是我的第一個 Python 腳本！
請輸入你的名字：小明
嗨，小明！歡迎來到 Python 的世界 🎉
```

### 🔄 兩者流程對比

```
Shell / Bash                    Python / VS Code
────────────────────            ────────────────────
1. 開終端機                      1. 安裝 Python + VS Code
2. 建立 hello.sh                2. 新增 hello.py
3. 寫入 #!/bin/bash + 指令       3. 寫入 Python 程式碼
4. chmod +x hello.sh            4. 點 ▶ 或輸入 python3 hello.py
5. ./hello.sh                   5. 結果顯示在下方 Terminal
```

> 💡 **新手最推薦：** 先用 VS Code 跑 Python，介面直覺、錯誤訊息清楚，上手最快！
