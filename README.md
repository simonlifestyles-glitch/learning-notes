## 🤔 什麼是程式語言？

程式語言是**你和電腦之間的溝通橋樑**。
 
| 角色 | 說明 |
|------|------|
| 人類 | 用程式語言寫「指令」 |
| 程式語言 | 中間的翻譯官 |
| 電腦 | 只懂 0 和 1，照指令執行 |

```
你寫的程式 → 翻譯器（編譯/執行） → 電腦執行（0和1） → 結果
```
 
例如你寫：
```python
print("你好")
```
電腦翻譯成機器碼後，畫面就會出現：`你好`

| 領域 | 能做什麼 | 使用語言 / 工具 | 連結 |

### 共同基礎

不管選哪個領域，都會碰到這三件事：

- **變數** — 儲存資料（例如：姓名、數字）
- **迴圈** — 重複執行某件事
- **函式** — 把一組指令包起來，可以重複使用

> 💡 不需要全部都學！找一個你有興趣的領域，從那裡開始就對了。
